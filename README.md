A mathematical text on the S4–Mamba line of neural state space sequence models.
It begins with a continuous-time state space model, discretises it for sampled
sequences, gives the state a structured memory, and follows that construction
through to selective and matrix-view models.

Read it at <https://ssm.guide>.

## Roadmap

The book is released as chapters are finished. Coming next are *Selective State
Spaces* (the Mamba block), *Matrix Views and State Space Duality* (Mamba-2), and
*Modern Refinements* (Mamba-3 and attention hybrids).
